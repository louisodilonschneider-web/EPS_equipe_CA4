# 🏫 Collège Pasteur — Portail EPS Numérique

Portail web centralisant les outils pédagogiques numériques du cours d'**Éducation Physique et Sportive** du Collège Pasteur.

---

## 🌐 Aperçu

Ce portail regroupe en une seule page l'ensemble des applications EPS développées pour les élèves et enseignants. Il permet d'accéder rapidement à chaque outil depuis n'importe quel appareil (PC, tablette, smartphone).

---

## 📁 Contenu du portail

### 🏸 Sport de Raquette — Badminton
**Lien :** [EPS Badminton](https://louisodilonschneider-web.github.io/EPS_Badminton/)

Outil permettant d'organiser des situations de jeu évolutives et des compétitions variées :

- 🎭 Matchs à thèmes
- 🏆 Ligues

### 🏃 Athlétisme — EPI Jean Bouin
**Lien :** [EPS Athlétisme](https://louisodilonschneider-web.github.io/EPS_Pasteur_Athletisme/)

Projet interdisciplinaire (EPI) centré sur la figure de **Jean Bouin**. Les élèves tentent de battre son record historique sur **3 000 m**, en combinant :
- 📏 Suivi des performances chronométriques
- 📖 Histoire du sport et de l'athlétisme
- 🔬 Démarche interdisciplinaire

---

- 📋 Composition d'enchaînements
- ⚖️ Initiation au jugement
- 🌟 Notation et lecture de code

## 🚀 Déploiement

Le portail est un fichier HTML statique autonome. Pour le mettre en ligne :

1. Créer un dépôt GitHub (ex. `EPS_Portail`)
2. Déposer le fichier `index.html` à la racine
3. Activer **GitHub Pages** dans les paramètres du dépôt (`Settings > Pages > Deploy from branch > main`)
4. Le portail est accessible à l'adresse : `https://<votre-username>.github.io/<nom-du-depot>/`

---

## ✏️ Personnalisation

Pour ajouter un nouveau site EPS, copier un bloc `.card` dans le fichier HTML et adapter :
- L'emoji et la couleur de thème
- Le titre et le sous-titre
- La description et les fonctionnalités
- Le lien `href` du bouton
